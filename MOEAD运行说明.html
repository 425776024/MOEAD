MOEAD运行说明

1.程序目录结构：

src：
  	problem：多目标测试问题函数目录
		ZDT1.py
		ZDT2.py
		ZDT3.py
		ZDT4.py
  	utils：算法及操作目录
		DE_Utils.py：差分算法操作
		Draw_Utils.py：绘图操作
		GA_Utils.py：遗传算法操作
		Mean_Vector_Util.py：均值向量生成
		MOEAD_Utils.py：MOEAD操作
 	vector_csv_file：测试函数权值向量
  		ZDT1.csv
		ZDT2.csv
		ZDT3.csv
		ZDT4.csv
	Main.py：主程序

2.程序运行说明：

Main.py 中__main__为启动函数：
if __name__ == '__main__':
    # np.random.seed(1)
    moead = MOEAD()
    moead.run()

3.参数配置

Test_fun = ZDT1 //选择测试函数，[ZDT1,ZDT2,ZDT3,ZDT4]
name = 'ZDT1'	//设置绘图标题，['ZDT1','ZDT2','ZDT3','ZDT4']
GA_DE_Utils = Utils.GA_Utils //选择执行算法，[Utils.DE_Utils,Utils.GA_Utils],基于遗传：Utils.GA_Utils；基于差分：Utils.DE_Utils
